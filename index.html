Citation Verifier

A lightweight tool for checking whether scholarly references actually exist. Paste a list in any prose-style format — APA, Chicago, Vancouver, MLA — and each entry is checked against the CrossRef database. Verified references are returned in clean, copy-ready form.

How it works. When a DOI is present, the tool resolves it directly via CrossRef. Otherwise, the bibliographic text is sent to CrossRef's search API and the closest match is scored by title-token overlap and year alignment. After verification, references are grouped by status and the canonical metadata returned by CrossRef is shown alongside each entry. You can copy individual references in APA / Vancouver / plain format, or export entire groups.
