Citation Verifier

A lightweight tool for checking whether scholarly references actually exist. Paste a list in any prose-style format — APA, Chicago, Vancouver, MLA — and each entry is checked against the CrossRef database.

No API key No sign-up DOIs preferred · titles matched heuristically

How it works. When a DOI is present, the tool resolves it directly via CrossRef. Otherwise, the bibliographic text is sent to CrossRef's search API and the closest match is scored by title-token overlap and year alignment. Statuses: Verified (high confidence), Possibly Real (partial match — manual check advised), Not Found (no record — possibly fabricated or not indexed).
